# Harry Potter Invisible Cloak

Make any colored cloth invisible!

## How to Run

**Option 1: Double-click the batch file**
```
run_harry.bat
```

**Option 2: Command line**
```powershell
python harry.py
```

## How to Use

1. **STEP 1**: Stay out of frame - Background will be captured automatically
2. **STEP 2**: Hold your cloth in the GREEN BOX and click on it
3. **STEP 3**: Wear the cloth - Watch it disappear! ✨

## Controls

- `+/-` - Adjust color tolerance
- `m` - Toggle mask view (debug)
- `s` - Save screenshot
- `q` - Quit

## What You Need

- Any solid colored cloth (RED works best!)
- Good lighting
- Webcam

No dataset needed - works with live camera!
